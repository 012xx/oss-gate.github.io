# oss-gate.github.io

OSS GateのWebサイトのリポジトリ。

https://oss-gate.github.io/

## 仕組み
GitHub pages（jekyll）を使用。markdownファイルとhtmlファイルを編集して、リポジトリにアップすると、GitHubがWebページに変換してくれる。

テーマには、defaultのminimaを使用。https://github.com/jekyll/minima

cssをsassで少し足している。
